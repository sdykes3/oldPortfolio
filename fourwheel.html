4 Wheeler Simulator

This is a simulation of driving an ATV, created in Unity using C#. The purpose was to create a vehicle simluation, from scratch, that feels like the real thing.

Focus was on making steering and jumps semi-realistic but still fun to control. A stabilizer was added to help the vehcile still be able to flip, but not so frequently or severely that it impacts play. Rough terrain allows jumping and climbing, but if attempting something too difficult the vehicle may flip onto its back and need to be reset.

Roles:
  • Idea concept
  • Vehicle controls
  • Camera movement
  • Gravity and collisions

Development:
  • Class project (individual)
  • Platform: Unity 3D
  • Language: C#
